 Для вызова функции надо просто написать её название и скобочки. В скобочки записываются аргументы функции. Например, print(variable): print - функция, variable - переменная.
2. Код в питоне чувствителен к табуляции. Это значит, что вместо обычных скобок, которые ограничивают "подкод" - используется табуляция.
3. Функции могут быть аргументами для других функций

| Operator | Name           | Description                                        |
| -------- | -------------- | -------------------------------------------------- |
| `a + b`  | Addition       | Sum of `a` and `b`                                 |
| `a - b`  | Subtraction    | Difference of `a` and `b`                          |
| `a * b`  | Multiplication | Product of `a` and `b`                             |
| `a / b`  | True division  | Quotient of `a` and `b`                            |
| `a // b` | Floor division | Quotient of `a` and `b`, removing fractional parts |
| `a % b`  | Modulus        | Integer remainder after division of `a` by `b`     |
| `a ** b` | Exponentiation | `a` raised to the power of `b`                     |
| `-a`     | Negation       | The negative of `a`                                |
**abs** - absolute value
**int** и **float** могут использоваться как функции для конвертации

Для определения функции используется **def**. Для описания собственноручно созданной функции можно использовать **docstings** - тройные кавычки

**return** - вывод функции. Без него функция ничего не вернёт. 

**sep** - разделитель при выводе, без него разделителями будут пробелы

Внутри функции можно задать стандартный аргумент:

```python fold title:Пример
def greet(who="Colin"):
    print("Hello,", who)
    
greet()
greet(who="Kaggle")
# (In this case, we don't need to specify the name of the argument, because it's unambiguous.)
greet("world")
```
В примере выше greet - функция с аргументом who, который, если не задано иного, равен Colin. В конце функции у нас идёт print - вывод. Тут нет return потому что мы не сохраняем значение при вызове, а сразу печатаем
Далее мы вызываем функцию и она сразу выводит нам строку

Так же можно использовать функции в качестве ключа для другой функции. Например, если задать функцию, которая будет возвращать остаток от деления на 5, то в таком случае можно будет задать эту функцию в качестве key-аргумента в функции, например, max()

**if elif else** 
**if** - если
**elif** - ..., иначе, если
**else** - иначе.
Грубо говоря, **if** - первый, **elif** - промежуточные, **else** - последний

Можно использовать небулевские значения в условии IF - функция будет использовать их как их булевский аналог (для цифр - все числа это True, кроме нуля; для строк - любые строки это True, кроме пустых строк)

Так же можно всякие условия писать просто так. Пример:
```python fold title:Example
number = 10
print(number<0) # Выведет False
```


### СПИСКИ
Список - упорядоченная очередность значений. Списки могут быть внутри списков. 
Индекс - указатель на n-ый элемент списка, начиная с нуля. Негативный индекс ведёт отчёт с конца, начиная с -1. 
Слайсинг - выборка определённого "куска" списка. Начало и конец для слайсинга указывать не обязательно
Конец слайсинга не будет включён в выборку
Функция len выдаёт длину списка
sorted - сортирует список
sum - складывает все значения списка вместе
min и max тоже работают со списками

### ОБЪЕКТЫ
Всё в питоне является объектом. Функции, которые прикреплены к объектам, называются **МЕТОД**. Не-функции будут называться **аттрибутами**.
