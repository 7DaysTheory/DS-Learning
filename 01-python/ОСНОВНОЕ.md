1. Для вызова функции надо просто написать её название и скобочки. В скобочки записываются аргументы функции. Например, print(variable): print - функция, variable - переменная.
2. Код в питоне чувствителен к табуляции. Это значит, что вместо обычных скобок, которые ограничивают "подкод" - используется табуляция.
3. Функции могут быть аргументами для других функций

| Operator | Name           | Description                                        |
| -------- | -------------- | -------------------------------------------------- |
| `a + b`  | Addition       | Sum of `a` and `b`                                 |
| `a - b`  | Subtraction    | Difference of `a` and `b`                          |
| `a * b`  | Multiplication | Product of `a` and `b`                             |
| `a / b`  | True division  | Quotient of `a` and `b`                            |
| `a // b` | Floor division | Quotient of `a` and `b`, removing fractional parts |
| `a % b`  | Modulus        | Integer remainder after division of `a` by `b`     |
| `a ** b` | Exponentiation | `a` raised to the power of `b`                     |
| `-a`     | Negation       | The negative of `a`                                |
**abs** - absolute value
**int** и **float** могут использоваться как функции для конвертации

Для определения функции используется **def**. Для описания собственноручно созданной функции можно использовать **docstings** - тройные кавычки

**return** - вывод функции. Без него функция ничего не вернёт. 

**sep** - разделитель при выводе, без него разделителями будут пробелы

Внутри функции можно задать стандартный аргумент:

```python fold title:Пример
def greet(who="Colin"):
    print("Hello,", who)
    
greet()
greet(who="Kaggle")
# (In this case, we don't need to specify the name of the argument, because it's unambiguous.)
greet("world")
```
В примере выше greet - функция с аргументом who, который, если не задано иного, равен Colin. В конце функции у нас идёт print - вывод. Тут нет return потому что мы не сохраняем значение при вызове, а сразу печатаем
Далее мы вызываем функцию и она сразу выводит нам строку

Так же можно использовать функции в качестве ключа для другой функции. Например, если задать функцию, которая будет возвращать остаток от деления на 5, то в таком случае можно будет задать эту функцию в качестве key-аргумента в функции, например, max()

**if elif else** 
**if** - если
**elif** - ..., иначе, если
**else** - иначе.
Грубо говоря, **if** - первый, **elif** - промежуточные, **else** - последний

Можно использовать небулевские значения в условии IF - функция будет использовать их как их булевский аналог (для цифр - все числа это True, кроме нуля; для строк - любые строки это True, кроме пустых строк)

Так же можно всякие условия писать просто так. Пример:

### СПИСКИ
Список - упорядоченная очередность значений. Списки могут быть внутри списков. 
Индекс - указатель на n-ый элемент списка, начиная с нуля. Негативный индекс ведёт отчёт с конца, начиная с -1. 
Слайсинг - выборка определённого "куска" списка. Начало и конец для слайсинга указывать не обязательно
Конец слайсинга не будет включён в выборку
Функция len выдаёт длину списка
sorted - сортирует список
sum - складывает все значения списка вместе
min и max тоже работают со списками

### ОБЪЕКТЫ
Всё в питоне является объектом. Функции, которые прикреплены к объектам, называются **МЕТОД**. Не-функции будут называться **аттрибутами**.

Всякое полезное про форматирование через f-строку.  
При форматировании можно указывать как переменные, так и числа/текст в явном виде.  
Число после `:` указывает на количество символов в ширину.  
Для наглядности использую символ `|`, чтобы визуализировать интервал:  
```python
number, text, width, char = 12345.6789, 'Python 3.10', 25, '-'  
print(f"Число {number =:13}") # Число number = 12345.6789  
print(f"|{'Python 3.10':25}|") # |Python 3.10 |  
print(f"|{text:{char}^{width}}|") # |-------Python 3.10-------|  
print(f"|{12345.6789:025}|") # |00000000000000012345.6789|  
print(f"|{number:{width}.2f}|") # | 12345.68|  
print(f"|{number:{char}<{width}f}|") # |12345.678900-------------|  
print(f"|{int(number):>{width}_d}|") # | 12_345|  
print(f"|{123456789:~^25,d}|") # |~~~~~~~123,456,789~~~~~~~|
```
                   
`<` Выравнивает по левому краю. У строк такое поведение по умолчанию  
`>` Выравнивает по правому краю. У чисел такое поведение по умолчанию  
`^` Выравнивает по центру  
По умолчанию символом заполнителем является пробел, но можно его заменить на символ, указанный перед символом выравнивания.  
Более того, чтобы добавить нули перед числом, 0 можно указать даже без символа выравнивания, например, `0{n}` добавит нули перед числом до ширины n, но другие символы необходимо указывать между `:` и `<`, `^`, или `>`.  
Также можно использовать форматирование для округления при выводе, разбиение на тысячи для чисел больше 1000.  
`.nf` Знаки после запятой (с округлением или прибавлением 0, если нужно)  
`f` (без .n) по умолчанию делает 6 знаков после запятой  
`,d` или `_d` Знаки разделителя `,` или `_` для чисел больше 1000  
Нельзя совмещать указание разделителя с указанием количества разрядов.
  
**Дополнение к f строкам**  
Знаки после запятой:  
```python
number_pi = 3.141592653589793  
a = 1 / 8  
b = 1 / 2  
c = 1 / 3  
print(f'{a=:.3f}') # a=0.125

print(f'{number_pi:6f}') # 3.141593  
print(f'{b=:.3f}') # b=0.500  
print(f'{c=:.3f}') # c=0.333  
print('------')  
print(f'{c:.3f}') # 0.333  
print(f'{c:.1f}') # 0.3  
print(f'{c:.2f}') # 0.33  
print(f'{c:.10f}') # 0.3333333333  
print('---')  
```

По умолчанию символом заполнителем является пробел, но можно его заменить на другое значение  
```python
number = 12345.6789  
print(f"Число {number=}")  
print(f"|{number:=<25}|")  
print(f"|{number:=>25}|")  
print(f"|{number:=^25}|")  
print('-' * 25)  
text = "Python 3.10"  
print(f"Строка {text=}")  
print(f"|{text:-<25}|")  
print(f"|{text:!>25}|")  
print(f"|{text:?^25}|")  
```

**Можно также влиять на знак разделителя между группами чисел:**  
```python
n = 12345678912345

print(f'{n:_d}') # 12_345_678_912_345  
sep = '_' # вложенная f-строка  
print(f'{n:{sep}d}') # 12_345_678_912_345
```

```python
s = 'Python' # индексы: 0,1,2,3,4,5 или -6,-5,-4,-3,-2,-1
```

базовые функции и операторы  

методы изменения регистра  
```python

s.upper() # все символы в верхний регистр
```

'----------------------'  
методы поиска  
```python
s.find('sub') # индекс первого вхождения, -1 если не найдено

s.index('sub') # как find(), но вызывает ошибку если не найдено  
s.rindex('sub') # то же, но с конца строки  
s.count('sub') # количество непересекающихся вхождений  
s.startswith('prefix') # True, если начинается с подстроки  
s.endswith('suffix') # True, если заканчивается подстрокой  
```
'----------------------'  
методы очистки  
```python
s.strip() # убрать пробелы с начала и конца
```

заметки  
• строки неизменяемы — методы возвращают новые строки  
• индексы начинаются с 0, отрицательные — с конца  
• методы is*() игнорируют небуквенные символы при проверке регистра  
• первые 128 символов Unicode = ASCII  
• нельзя сравнивать строки с числами: '5' > 4 → ошибка  
• методы find/rfind безопаснее index/rindex (не вызывают ошибку)
