### 📖 About this repo

This is where I keep every notebook, script, and project from my Data Science learning roadmap. No bootcamp, no degree — just a structured plan and consistent practice, documented as I go.

**A note on data:** I work as a DWH analyst in fintech, and my company's security policy doesn't allow using any work-related data here. Everything in this repo is built exclusively on public, open datasets (Kaggle, UCI, etc.).

### 🛠️ Setup

```bash
git clone https://github.com/7DaysTheory/DS-Learning.git
cd DS-Learning
python -m venv venv
source venv/bin/activate   # Windows: venv\Scripts\activate
pip install jupyterlab pandas numpy matplotlib
```

A full `requirements.txt` will be added as the project list grows.
