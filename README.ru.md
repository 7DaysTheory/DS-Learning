### 📖 О репозитории

Здесь собраны все ноутбуки, скрипты и проекты из моего плана обучения Data Science. Без буткемпа и без профильного образования — только структурированный план и регулярная практика, документируемая по ходу пути.

**Важное замечание про данные:** я работаю DWH-аналитиком в финтехе, и политика информационной безопасности компании не позволяет использовать здесь какие-либо рабочие данные. Все проекты построены исключительно на открытых публичных датасетах (Kaggle, UCI и т.д.).

### 🛠️ Настройка окружения

```bash
git clone https://github.com/7DaysTheory/DS-Learning.git
cd DS-Learning
python -m venv venv
source venv/bin/activate   # Windows: venv\Scripts\activate
pip install jupyterlab pandas numpy matplotlib
```

Полный `requirements.txt` появится по мере роста списка проектов.
